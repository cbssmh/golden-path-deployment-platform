# v0.1.0 — Reproducible GitOps Foundation

## Release purpose

Establish a small, reproducible local Kubernetes deployment path driven by GitOps.

## Implemented

kind bootstrap and teardown, Argo CD installation, Root Application bootstrap,
a Kustomize Service A dev deployment, validation scripts, CI definitions,
documentation, and an evidence layout.

## Validated

Static validation is intended to verify shell syntax, Kustomize rendering,
policy checks, and whitespace. Runtime validation is defined in the rebuild
runbook and depends on Docker, kind, and a reachable public GitOps repository.

## Scope, exclusions, and limitations

Only the v0.1.0 scope in `docs/scope-and-non-goals.md` is included. Private
GitOps repositories, automatic image updates, cloud deployments, and
production-operation capabilities are intentionally absent.

## Next version

No v0.2.0 feature is implemented or committed by this draft.
